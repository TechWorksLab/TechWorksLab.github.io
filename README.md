# TechWorksLab.github.io

Marketing site for **TechWorksLab** — a technology provider for pharmaceutical and life-sciences companies.

We build next-generation, open-source-first software that accelerates **drug development, clinical trial analytics, CDISC data standards (SDTM/ADaM), TLF/TLG automation, and FDA/regulatory submissions** using validated R & Python and the [pharmaverse](https://pharmaverse.org/) ecosystem.

## Stack
Static GitHub Pages site — `index.html`, `styles.css`, `script.js`. No build step.

## Sections
- **Solutions** — clinical analytics, SDTM/ADaM, TLF automation, submissions, GxP SCE, GenAI
- **Open Source** — the pharmaverse + R Submissions ("r4sub") FDA pilots
- **Software** — our products (CliniTransformR, ModelBox, Data Extractor, autotestpkg, Rmarker, ScrapText, and more)
- **Submissions** — the data-to-eCTD workflow
- **Technology** — R/pharmaverse + Python stack, validation & compliance
