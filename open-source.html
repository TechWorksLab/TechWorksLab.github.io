Open Source

We build with the open clinical software community

The packages that produce clinical datasets, reports, and submissions are increasingly open and shared across the industry. We adopt them, contribute to them, and qualify them for regulated work.

The pharmaverse

A curated set of packages covering the path from raw data to clinical reporting. It lets sponsors collaborate on shared tooling instead of each rebuilding the same components.

R submission pilots

The R Consortium submission working group has run a series of pilots with the FDA, delivering tables, datasets, and interactive review applications through the electronic gateway.

Contribution and qualification

We report issues, submit fixes upstream, and run the validation needed to use these packages safely inside audit ready workflows.
