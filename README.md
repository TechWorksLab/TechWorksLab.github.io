# TechWorksLab.github.io

Marketing site for **TechWorksLab**, a technology partner for pharmaceutical, clinical research, and life sciences organizations.

We build validated platforms for clinical trial analytics, CDISC data standards (SDTM and ADaM), reporting (tables, listings, figures), and regulatory submissions.

## Stack
Static GitHub Pages site. Plain HTML, one shared `styles.css`, and a small `script.js` for navigation. No build step.

## Pages
- `index.html` — Home
- `solutions.html` — Solutions across the drug development lifecycle
- `software.html` — Proprietary software platforms
- `open-source.html` — The open clinical software ecosystem (pharmaverse, R submission pilots)
- `about.html` — Company
- `insights.html` — Articles listing
- `insights-adam-admiral.html`, `insights-r-submissions.html`, `insights-validated-sce.html` — Articles
- `contact.html` — Contact

## Conventions
- No em dashes or en dashes in copy.
- No AI or machine learning references.
- Header and footer markup is duplicated across pages (static site, no includes); update all pages when changing them.
